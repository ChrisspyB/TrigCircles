Sinusoids and Circles

Every sinusoidal function can be represented by a corresponding circle. By tracing the our height as we travel along the circumference of a circle, we produce a sine wave of period 2π, sin(x).
Click on the diagram to draw a wave.
If were to instead start tracing from the top of the circle (90° or 0.5π later) the same procedure produces a cosine wave, cos(x).
Where we start drawing the wave is its initial phase. The above diagram demonstrates that cos(x) and sin(x) have a phase difference of 0.5π radians i.e. cos(x) = sin(x+0.5π).

By adjusting the circle's radius, our starting position and how quickly we trace the circle we can plot arbitrary sinusoidal functions of the form Asin(ωx+φ). Fiddle with the sliders below, and watch how it affects both the circle and waveform.

Sums of Circles

If an arbitrary sine function has a corresponding circle, it follows that a sum of two sine waves can be represented by a corresponding sum of circles.
Click to draw one period, hit SCAN to keep drawing.

Fourier Series

As it turns out, any periodic function can be expressed as an infinite sum of sin and cos functions of different amplitudes and frequencies. As such, any periodic function can be visualised with an infinite number of circles! We can't really draw an infinite number of circles, but we can approximate by drawing the first few in the series.
Try adding terms while in SCAN mode and watch as the waveform begins to take shape.

Travelling Waves

So far we have only considered functions of a single variable, x. Our circles can model how a single atom's position varies with time, or a how vibration in a string varies with position. If we want to consider waves which vary with time and position, a travelling wave, each point in space needs to have its own set of circles which oscillate in time.
